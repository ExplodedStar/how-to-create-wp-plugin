Описание того как создать плагин для WordPress

1. Все плагины лежат в папке ../wp-content/plugins
2. Создаем директорию (папку) с Вашим плагином ../wp-content/plugins/название-плагина
3. Заполняем информация о плагине в главном файле плагина. Сторока "Plugin Name" обязательна 
    <?php
	/*
	Plugin Name: Название плагина
	Plugin URI: http://страница_с_описанием_плагина_и_его_обновлений
	Description: Краткое описание плагина.
	Version: Номер версии плагина, например: 1.0
	Author: Имя автора плагина
	Author URI: http://страница_автора_плагина
	*/
	?>
